Various ways to use Insert statement in PL/SQL

AIn Oracle PL/SQL, inserting data into tables can be achieved through various types of INSERT statements, each serving different purposes. Here are some key types:
1. Single Row Insert: Adds a single row to a table.
2. Multi-Row Insert: Inserts multiple rows selected from another source.
3. Insert All: Allows inserting multiple rows into one or more tables in a single statement.
4. Insert First: Inserts rows based on the first condition that evaluates to true.
5. Insert with Subquery: Uses a subquery to determine the values to insert.
